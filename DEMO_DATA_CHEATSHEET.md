Q: Demo data not showing up?
A: Check environment variable:
   $ echo $SEED_DEMO_DATA
   Should output: true
   
   If not set:
   $ export SEED_DEMO_DATA=true
   $ python -m uvicorn app.main:app --reload

Q: Cannot connect to databases?
A: Verify connections in .env:
   DATABASE_URL=postgresql://user:pass@localhost:5432/lending_db
   MONGODB_URI=mongodb://localhost:27017/lending
   
   Test MongoDB:
   $ mongosh mongodb://localhost:27017
   
   Test PostgreSQL:
   $ psql -U user -h localhost -d lending_db -c "SELECT 1"

Q: Getting duplicate records?
A: The seeder is idempotent (won't create duplicates)
   If you see duplicates:
   $ docker-compose -f docker-compose.demo.yml down -v
   $ docker-compose -f docker-compose.demo.yml up --build

Q: How to customize demo data?
A: Edit the demo_seeder.py file:
   $ vi lending-mvp/backend/app/utils/demo_seeder.py
   
   Modify these constants:
   - SAMPLE_BRANCHES
   - SAMPLE_USERS
   - SAMPLE_CUSTOMERS_*
   - SAMPLE_LOAN_PRODUCTS
   - etc.
   
   Then re-run with SEED_DEMO_DATA=true

Q: How to reset and start fresh?
A: Docker:
   $ docker-compose -f docker-compose.demo.yml down -v
   $ docker-compose -f docker-compose.demo.yml up --build
   
   Python:
   Clear MongoDB and PostgreSQL manually, then:
   $ export SEED_DEMO_DATA=true
